# Banking Transaction Analytics

## Overview
This project analyzes banking transaction data to identify spending trends, high-value transactions, and potential fraud risks.

## Tools Used
- Python (Pandas)
- SQL
- Power BI

## Key Analysis
- Credit vs Debit trends
- High-value transaction detection
- Risk-based transaction analysis
- Monthly transaction trends
- Customer spending insights
